To use, copy the three files into the "Binary" folder where the exe is located.
For it to work in the editor, copy them to where Unreal is installed. It should be something like: C:\Program Files\Epic Games\UE_4.16\Engine\Binaries\Win64

To adjust the input mapping, launch x360ce_x64 before starting the simulator.
Modify the mapping, save and close the window.
